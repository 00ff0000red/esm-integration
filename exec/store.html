Store Validity¶

Typing¶

The following typing rules specify when a runtime function, table, memory, global or module is well-typed, relative to a store \(S\) in which they live.

Function Instance \(\{\href{../exec/runtime.html#syntax-funcinst}{\mathsf{type}}~\href{../syntax/types.html#syntax-functype}{\mathit{functype}}, \href{../exec/runtime.html#syntax-funcinst}{\mathsf{module}}~\href{../exec/runtime.html#syntax-moduleinst}{\mathit{moduleinst}}, \href{../exec/runtime.html#syntax-funcinst}{\mathsf{code}}~\href{../syntax/modules.html#syntax-func}{\mathit{func}}\}\)¶

\[\frac{ \vdash \href{../syntax/types.html#syntax-functype}{\mathit{functype}} \mathrel{\mbox{ok}} \qquad S \vdash \href{../exec/runtime.html#syntax-moduleinst}{\mathit{moduleinst}} : C \qquad C \vdash \href{../syntax/modules.html#syntax-func}{\mathit{func}} : \href{../syntax/types.html#syntax-functype}{\mathit{functype}} }{ S \vdash \{\href{../exec/runtime.html#syntax-funcinst}{\mathsf{type}}~\href{../syntax/types.html#syntax-functype}{\mathit{functype}}, \href{../exec/runtime.html#syntax-funcinst}{\mathsf{module}}~\href{../exec/runtime.html#syntax-moduleinst}{\mathit{moduleinst}}, \href{../exec/runtime.html#syntax-funcinst}{\mathsf{code}}~\href{../syntax/modules.html#syntax-func}{\mathit{func}}\} : \href{../syntax/types.html#syntax-functype}{\mathit{functype}} }\]

Table Instance \(\{ \href{../exec/runtime.html#syntax-tableinst}{\mathsf{elem}}~(\mathit{fa}^?)^n, \href{../exec/runtime.html#syntax-tableinst}{\mathsf{max}}~m^? \}\)¶

\[\frac{ (S \vdash \mathit{fa}^? \mathrel{\mbox{ok}})^n \qquad \vdash \{\href{../syntax/types.html#syntax-limits}{\mathsf{min}}~n, \href{../syntax/types.html#syntax-limits}{\mathsf{max}}~m^?\} \mathrel{\mbox{ok}} }{ S \vdash \{ \href{../exec/runtime.html#syntax-tableinst}{\mathsf{elem}}~(\mathit{fa}^?)^n, \href{../exec/runtime.html#syntax-tableinst}{\mathsf{max}}~m^? \} : \{\href{../syntax/types.html#syntax-limits}{\mathsf{min}}~n, \href{../syntax/types.html#syntax-limits}{\mathsf{max}}~m^?\}~\href{../syntax/types.html#syntax-elemtype}{\mathsf{anyfunc}} }\]
\[\frac{ }{ S \vdash \epsilon \mathrel{\mbox{ok}} } \qquad \frac{ \href{../exec/runtime.html#syntax-funcaddr}{\mathit{funcaddr}} < |S.\href{../exec/runtime.html#syntax-store}{\mathsf{funcs}}| }{ S \vdash \href{../exec/runtime.html#syntax-funcaddr}{\mathit{funcaddr}} \mathrel{\mbox{ok}} }\]

Memory Instance \(\{ \href{../exec/runtime.html#syntax-meminst}{\mathsf{data}}~b^n, \href{../exec/runtime.html#syntax-meminst}{\mathsf{max}}~m^? \}\)¶

\[\frac{ \vdash \{\href{../syntax/types.html#syntax-limits}{\mathsf{min}}~n, \href{../syntax/types.html#syntax-limits}{\mathsf{max}}~m^?\} \mathrel{\mbox{ok}} }{ S \vdash \{ \href{../exec/runtime.html#syntax-meminst}{\mathsf{data}}~b^n, \href{../exec/runtime.html#syntax-meminst}{\mathsf{max}}~m^? \} : \{\href{../syntax/types.html#syntax-limits}{\mathsf{min}}~n, \href{../syntax/types.html#syntax-limits}{\mathsf{max}}~m^?\} }\]

Global Instance \(\{ \href{../exec/runtime.html#syntax-globalinst}{\mathsf{value}}~(t.\href{../syntax/instructions.html#syntax-instr-numeric}{\mathsf{const}}~c), \href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}}~\href{../syntax/types.html#syntax-mut}{\mathit{mut}} \}\)¶

\[\frac{ }{ S \vdash \{ \href{../exec/runtime.html#syntax-globalinst}{\mathsf{value}}~(t.\href{../syntax/instructions.html#syntax-instr-numeric}{\mathsf{const}}~c), \href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}}~\href{../syntax/types.html#syntax-mut}{\mathit{mut}} \} : \href{../syntax/types.html#syntax-mut}{\mathit{mut}}~t }\]

Export Instance \(\{ \href{../exec/runtime.html#syntax-exportinst}{\mathsf{name}}~\href{../syntax/values.html#syntax-name}{\mathit{name}}, \href{../exec/runtime.html#syntax-exportinst}{\mathsf{value}}~\href{../exec/runtime.html#syntax-externval}{\mathit{externval}} \}\)¶

\[\frac{ S \vdash \href{../exec/runtime.html#syntax-externval}{\mathit{externval}} : \href{../syntax/types.html#syntax-externtype}{\mathit{externtype}} }{ S \vdash \{ \href{../exec/runtime.html#syntax-exportinst}{\mathsf{name}}~\href{../syntax/values.html#syntax-name}{\mathit{name}}, \href{../exec/runtime.html#syntax-exportinst}{\mathsf{value}}~\href{../exec/runtime.html#syntax-externval}{\mathit{externval}} \} \mathrel{\mbox{ok}} }\]

Module Instance¶

\[\begin{split}\frac{ \begin{array}{@{}c@{}} (\vdash \href{../syntax/types.html#syntax-functype}{\mathit{functype}} \mathrel{\mbox{ok}})^\ast \\ (S \vdash S.\href{../exec/runtime.html#syntax-store}{\mathsf{funcs}}[\href{../exec/runtime.html#syntax-funcaddr}{\mathit{funcaddr}}] : \href{../syntax/types.html#syntax-functype}{\mathit{functype}}')^\ast \qquad (S \vdash S.\href{../exec/runtime.html#syntax-store}{\mathsf{tables}}[\href{../exec/runtime.html#syntax-tableaddr}{\mathit{tableaddr}}] : \href{../syntax/types.html#syntax-tabletype}{\mathit{tabletype}})^\ast \\ (S \vdash S.\href{../exec/runtime.html#syntax-store}{\mathsf{mems}}[\href{../exec/runtime.html#syntax-memaddr}{\mathit{memaddr}}] : \href{../syntax/types.html#syntax-memtype}{\mathit{memtype}})^\ast \qquad (S \vdash S.\href{../exec/runtime.html#syntax-store}{\mathsf{globals}}[\href{../exec/runtime.html#syntax-globaladdr}{\mathit{globaladdr}}] : \href{../syntax/types.html#syntax-globaltype}{\mathit{globaltype}})^\ast \\ (S \vdash \href{../exec/runtime.html#syntax-exportinst}{\mathit{exportinst}} \mathrel{\mbox{ok}})^\ast \qquad (\href{../exec/runtime.html#syntax-exportinst}{\mathit{exportinst}}.\href{../exec/runtime.html#syntax-exportinst}{\mathsf{name}})^\ast ~\mbox{disjoint} \\ C = \{\href{../valid/conventions.html#context}{\mathsf{types}}~\href{../syntax/types.html#syntax-functype}{\mathit{functype}}^\ast, \href{../valid/conventions.html#context}{\mathsf{funcs}}~{\href{../syntax/types.html#syntax-functype}{\mathit{functype}}'}^\ast, \href{../valid/conventions.html#context}{\mathsf{tables}}~\href{../syntax/types.html#syntax-tabletype}{\mathit{tabletype}}^\ast, \href{../valid/conventions.html#context}{\mathsf{mems}}~\href{../syntax/types.html#syntax-memtype}{\mathit{memtype}}^\ast, \href{../valid/conventions.html#context}{\mathsf{globals}}~\href{../syntax/types.html#syntax-globaltype}{\mathit{globaltype}}^\ast\} \end{array} }{ S \vdash \{ \begin{array}[t]{@{}l@{~}l@{}} \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{types}} & \href{../syntax/types.html#syntax-functype}{\mathit{functype}}^\ast, \\ \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{funcaddrs}} & \href{../exec/runtime.html#syntax-funcaddr}{\mathit{funcaddr}}^\ast, \\ \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{tableaddrs}} & \href{../exec/runtime.html#syntax-tableaddr}{\mathit{tableaddr}}^\ast, \\ \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{memaddrs}} & \href{../exec/runtime.html#syntax-memaddr}{\mathit{memaddr}}^\ast, \\ \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{globaladdrs}} & \href{../exec/runtime.html#syntax-globaladdr}{\mathit{globaladdr}}^\ast \\ \href{../exec/runtime.html#syntax-moduleinst}{\mathsf{exports}} & \href{../exec/runtime.html#syntax-exportinst}{\mathit{exportinst}}^\ast ~\} : C \end{array} }\end{split}\]

Store¶

\[\begin{split}\frac{ \begin{array}{@{}c@{}} (S \vdash \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}} : \href{../syntax/types.html#syntax-functype}{\mathit{functype}}')^\ast \qquad (S \vdash \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}} : \href{../syntax/types.html#syntax-tabletype}{\mathit{tabletype}})^\ast \\ (S \vdash \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}} : \href{../syntax/types.html#syntax-memtype}{\mathit{memtype}})^\ast \qquad (S \vdash \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}} : \href{../syntax/types.html#syntax-globaltype}{\mathit{globaltype}})^\ast \\ S = \{ \href{../exec/runtime.html#syntax-store}{\mathsf{funcs}}~\href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}^\ast, \href{../exec/runtime.html#syntax-store}{\mathsf{tables}}~\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}^\ast, \href{../exec/runtime.html#syntax-store}{\mathsf{mems}}~\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}^\ast, \href{../exec/runtime.html#syntax-store}{\mathsf{globals}}~\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}^\ast \} \end{array} }{ \vdash S \mathrel{\mbox{ok}} }\end{split}\]

Extension¶

Function Instance¶

  • A function instance must remain unchanged.
\[\frac{ \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_1 = \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_2 }{ \vdash \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_1 \prec \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_2 }\]

Table Instance¶

  • The length of \(|\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{elem}}|\) must not shrink.
  • The value of \(\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{max}}\) must remain unchanged.
\[\frac{ |\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_1.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{elem}}| \leq |\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_2.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{elem}}| \qquad \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_1.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{max}} = \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_2.\href{../exec/runtime.html#syntax-tableinst}{\mathsf{max}} }{ \vdash \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_1 \prec \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_2 }\]

Memory Instance¶

  • The length of \(|\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}.\href{../exec/runtime.html#syntax-meminst}{\mathsf{data}}|\) must not shrink.
  • The value of \(\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}.\href{../exec/runtime.html#syntax-meminst}{\mathsf{max}}\) must remain unchanged.
\[\frac{ |\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_1.\href{../exec/runtime.html#syntax-meminst}{\mathsf{data}}| \leq |\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_2.\href{../exec/runtime.html#syntax-meminst}{\mathsf{data}}| \qquad \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_1.\href{../exec/runtime.html#syntax-meminst}{\mathsf{max}} = \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_2.\href{../exec/runtime.html#syntax-meminst}{\mathsf{max}} }{ \vdash \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_1 \prec \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_2 }\]

Global Instance¶

  • The mutability \(\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}}\) must remain unchanged.
  • If \(\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}}\) is \(\href{../syntax/types.html#syntax-mut}{\mathsf{const}}\), then the value\(\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{value}}\) must also remain unchanged.
\[\begin{split}\frac{ \begin{array}{@{}c@{}} \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{value}} = \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{value}} \\ \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}} = \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}} = \href{../syntax/types.html#syntax-mut}{\mathsf{const}} \end{array} }{ \vdash \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1 \prec \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2 } \qquad \frac{ \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}} = \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2.\href{../exec/runtime.html#syntax-globalinst}{\mathsf{mut}} = \href{../syntax/types.html#syntax-mut}{\mathsf{var}} }{ \vdash \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1 \prec \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2 }\end{split}\]

Store¶

\[\begin{split}\frac{ \begin{array}{@{}ccc@{}} S_1.\href{../exec/runtime.html#syntax-store}{\mathsf{funcs}} = \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_1^\ast & S_2.\href{../exec/runtime.html#syntax-store}{\mathsf{funcs}} = {\href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}'_1}^\ast~\href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_2^\ast & (\href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}_1 \prec \href{../exec/runtime.html#syntax-funcinst}{\mathit{funcinst}}'_1)^\ast \\ S_1.\href{../exec/runtime.html#syntax-store}{\mathsf{tables}} = \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_1^\ast & S_2.\href{../exec/runtime.html#syntax-store}{\mathsf{tables}} = {\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}'_1}^\ast~\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_2^\ast & (\href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}_1 \prec \href{../exec/runtime.html#syntax-tableinst}{\mathit{tableinst}}'_1)^\ast \\ S_1.\href{../exec/runtime.html#syntax-store}{\mathsf{mems}} = \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_1^\ast & S_2.\href{../exec/runtime.html#syntax-store}{\mathsf{mems}} = {\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}'_1}^\ast~\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_2^\ast & (\href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}_1 \prec \href{../exec/runtime.html#syntax-meminst}{\mathit{meminst}}'_1)^\ast \\ S_1.\href{../exec/runtime.html#syntax-store}{\mathsf{globals}} = \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1^\ast & S_2.\href{../exec/runtime.html#syntax-store}{\mathsf{globals}} = {\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}'_1}^\ast~\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_2^\ast & (\href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}_1 \prec \href{../exec/runtime.html#syntax-globalinst}{\mathit{globalinst}}'_1)^\ast \\ \end{array} }{ \vdash S_1 \prec S_2 }\end{split}\]
